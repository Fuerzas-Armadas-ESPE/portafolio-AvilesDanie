Generador de órdenes para el restaurante MegaApanados.

Proyecto desarrollado en Java con el IDE Apache NetBeans 15. El proyecto se enfoca en generar una interfaz que da al usuario opciones relacionadas con el servicio automatizado del restaurante, permitiéndole ver el menú, selección de platillos y generar su orden con su respectivo comprobante, toda la información generada y platillos disponibles son guardados y extraídos desde la base de datos de MongoDB. Adicionalmente se agregó la opción de envió de comentarios por parte del cliente como medida para conocer posibles mejoras.
